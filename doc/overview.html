Overview

Report creation on a paper oriented media is a major concern for any PHP developer. Understanding that the offerings weren't variate, InterAKT Online started the development of an Open Source PS generator. Using only native PHP function and requiring only the ubiquitous GhostScript engine, Yaps aim in becoming a very powerful library for PHP developers. YaPs has now already implemented all important functions for standard report generation and many other functions are on their way. YaPs license is [10]GPL, because we want to share it with the open source community (we are using in production environments other GPLed softwares and we are very pleased of them). We hope the developer community will help us improve this library. YaPs is the core library beneath PDFReports and PDFreportsLite.
